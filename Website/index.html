If you're facing client side issues while trying to get into the game please do these steps:

Issue 1: Game continously redirects you to the homepage.

Solution: Turn off enhanced tracking protection in your browser. This allows your browser to use cross-site cookies.

Issue 2: Game isn't loading in assets

Solution: The server isn't on. Please notify a server admin to start the server, and refresh the game.
